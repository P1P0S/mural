<h3>Para rodar localmente:</h3>

```
git clone https://github.com/P1P0S/mural.git
```

<h3>Instalando todas as dependencias:</h3>

```
cd mural
npm install
```

<h3>Rodando o servidor utilizando o <a href="https://nodemon.io">Nodemon</a>:
</h3>

```
npx nodemon
```
